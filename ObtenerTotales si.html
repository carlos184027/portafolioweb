  • 1-¿Qué es Big Data?
    • Big data es una técnica para reducir el tamaño de los conjuntos de datos, lo cual facilita su almacenamiento y análisis en sistemas informáticos convencionales.
    Big data es un método para predecir el futuro con total certeza basado en el análisis de grandes cantidades de datos históricos.
    Big Data son conjuntos de datos cuyo tamaño o tipo tiene una capacidad superior para capturar, gestionar y procesar los datos
    Big data es un término utilizado para describir una serie de algoritmos de inteligencia artificial diseñados específicamente para trabajar con conjuntos de datos pequeños y medianos.

  • 2-¿Qué permite el analisis de datos?
    • El análisis de datos permite a los analistas, investigadores y usuarios de negocio mejorar y agilizar las decisiones, utilizando datos que antes eran inaccesibles o inutilizables.
    El análisis de datos permite la manipulación directa de la realidad física, alterando eventos en el mundo real a través de la modificación de los datos.
    El análisis de datos permite la creación de universos alternativos donde las leyes de la física y la lógica pueden ser reescritas a voluntad.
    El análisis de datos permite la comunicación telepática entre individuos, transmitiendo información directamente desde las mentes de una persona a otra sin necesidad de palabras o señales físicas.
